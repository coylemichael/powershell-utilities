## [1.0.0] - 2025-10-31

### Added
- Initial repository structure with Teams/ExternalAccess organization
- **Install-TeamsModule.ps1**: Install and validate MicrosoftTeams PowerShell module
- **Connect-TeamsTenant.ps1**: Establish authenticated connection to Microsoft Teams
- **Validate-FederationConfig.ps1**: Audit Teams federation configuration and policy mode
- **Enable-AllowListMode.ps1**: Switch Teams federation to allow-list mode (zero-trust)
- **Manage-AllowListDomains.ps1**: Interactive menu tool for managing allowed/blocked domains
- README files at root, Teams, and ExternalAccess levels with MVP documentation
